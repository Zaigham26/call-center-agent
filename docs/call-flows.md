# Call Flows (Thai + English) — Tooth Care Bangkok

## 0) Global rules
- Mirror language (TH/EN). If unclear, ask.
- Don’t diagnose; don’t guarantee outcomes.
- Use price range only: 5,000–10,000 THB (depends on exam).
- Scheduling window: Mon–Sat 09:00–17:00 (Asia/Bangkok). Closed Sunday.
- Add 10-minute buffer after each appointment block.
- Escalate emergencies (breathing trouble, uncontrolled bleeding, rapidly worsening swelling, fever with swelling/pain, major trauma, pain 9–10/10).

---

## 1) New inquiry → book appointment
### Goal
Convert inquiry into a confirmed Google Calendar appointment.

### Steps
1. Greet + confirm language (if needed).
2. Ask service type: Implant / Aligner / Cosmetic / Other.
3. Ask main concern.
4. Triage:
   - Pain score 0–10
   - Swelling? Fever? Trauma?
   - If emergency triggers → switch to Flow 6.
5. Collect booking fields:
   - Name, phone, channel preference
6. Ask timing:
   - If “today/now” → immediate booking rule (earliest slot >= 30 mins from now).
   - Else ask for 2 preferred date/time options.
7. Offer 1–3 available slots (or confirm earliest slot for immediate).
8. Confirm booking:
   - Summarize name/service/date-time/location map link.
9. Send confirmation message + reminder to bring ID and arrive early (optional).

---

## 2) Price question → convert to booking
### Goal
Answer price question safely and move to consult booking.

### Steps
1. Provide range: 5,000–10,000 THB.
2. State dependency on exam / case.
3. Offer consultation booking and proceed to Flow 1.

---

## 3) Reschedule appointment
### Steps
1. Ask name + phone used to book.
2. Ask preferred new date/time (2 options) or “earliest available”.
3. Find availability; propose 3 slots.
4. Confirm reschedule + send updated confirmation.

---

## 4) Cancel appointment (free cancellation)
### Steps
1. Ask name + phone used to book.
2. Confirm cancellation.
3. Offer to rebook another time.

---

## 5) Directions / hours / location
### Steps
1. Provide hours: Mon–Sat 09:00–17:00, closed Sunday.
2. Provide map link: https://maps.app.goo.gl/ySh5ARMpEKNfaUKP9
3. Offer booking.

---

## 6) Urgent symptoms triage / escalation
### Triggers
- Trouble breathing/swallowing
- Uncontrolled bleeding
- Rapidly worsening facial swelling
- High fever with swelling/pain
- Major trauma/accident with tooth displacement
- Pain 9–10/10 not controlled

### Steps
1. Empathize briefly.
2. If severe trigger present: advise emergency care immediately.
3. Ask for phone number + symptoms summary.
4. Escalate to staff (define your internal process).
5. If they still want to come today, book earliest slot available, but still advise emergency care if severe.

---

## 7) Walk-in / “Can I come now?”
### Steps
1. Collect name + phone.
2. Confirm service type + quick triage (pain score, swelling, fever, trauma).
3. Book earliest slot >= 30 minutes from now (if within hours).
4. If outside hours or Sunday: offer next available day/time.
